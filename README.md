# DDDM · Модуль 5 — тестовый прогон для автономного агента

Публичный self-contained репозиторий, на котором **автономный агент проходит весь
Модуль 5** курса «От данных к решению» — от разминочных разрезов до флагманского
Claw-like кейса с файлом-отчётом. Нужен, чтобы протестировать агента-исполнителя как
запасной вариант, пока готовится «взрослая» версия с доступом к GitLab.

## Как пользоваться (человеку)

1. Дай агенту ссылку на этот репозиторий и на **[`HANDOFF.md`](HANDOFF.md)** —
   больше ничего.
2. Укажи режим одним словом: **`пилот`** (с твоим участием), **`ассистент`**
   (подтверждения на контрольных точках) или **`автопилот`** (агент проходит всё сам).
   Если не указать — агент идёт в **автопилоте**.
3. Дождись, пока агент сложит артефакты в `run/` и отдаст `run/PASSPORT.md` — отчёт
   о прохождении.

## Что внутри

| Папка | Что |
|-------|-----|
| **`HANDOFF.md`** | Главная инструкция: агент читает её и проходит модуль по шагам. Три режима. |
| `data/` | Все данные (заказы + «грязные» расходы + генераторы + SQL-дамп). Внешних доступов не нужно. |
| `program/` | Карта модуля: уроки и сквозная линия. |
| `demos/` | Справочные реализации уроков 5.1–5.8 (миниатюра кодинг-агента). |
| `reference_solution/` | Эталон для проверки человеком — агент открывает его только на самопроверке. |

## Требования

Python 3 + `pandas matplotlib numpy openpyxl`. Ключ LLM-роутера (`AI_PROVIDER_TOKEN`)
**не обязателен** — основную работу агент делает своим кодом; ключ нужен лишь для
опционального запуска демо-скриптов «живьём».

## Источник
